Ubuntu16.04下JDK安装配置

JDK 安装

  • 将 jdk-8u172-linux-x64.tar.gz，放到你想安装的目录
  • tar -zxvf jdk-8u172-linux-x64.tar.gz 解压文件
  • sudo vim ~/.bashrc 打开文件，配置环境变量
  • source ~/.bashrc 立即生效
  • sudo update-alternatives –install /usr/bin/java java /home/cxp/java/jdk1.8.0_172/bin/java 300
  • sudo update-alternatives –install /usr/bin/javac javac /home/cxp/java/jdk1.8.0_172/bin/javac 300
  • sudo update-alternatives –config java
  • java -version
