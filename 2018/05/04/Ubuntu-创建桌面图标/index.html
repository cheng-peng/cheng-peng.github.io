Ubuntu 创建桌面图标

创建桌面启动图标

  • sudo touch sublime.desktop 创建文件sublime.desktop文件
  • 移动到/usr/share/applications/文件夹下,或者直接创建在此文件夹下
  • sudo vim sublime.desktop 编写sublime.desktop文件
  • sudo chmod 777 sublime.desktop 设置权限
