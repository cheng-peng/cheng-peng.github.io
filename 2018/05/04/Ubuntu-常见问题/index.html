Ubuntu 常见问题

ubuntu 关机时喇叭“破音

  • sudo gedit /usr/lib/pm-utils/power.d/intel-audio-powersave
  • 注释这一行：INTEL_AUDIO_POWERSAVE=${INTEL_AUDIO_POWERSAVE:-true}
  • 加入这句话：INTEL_AUDIO_POWERSAVE=true

无法获得锁 /var/lib/dpkg/lock -open （11：资源暂时不可用）

  • 解决方法，强制解除：
  • sudo rm /var/cache/apt/archives/lock
  • sudo rm /var/lib/dpkg/lock

解决apt-get 下载速度慢

  • sudo add-apt-repository ppa:saiarcot895/myppa
  • sudo apt-get update
  • sudo apt-get -y install apt-fast
  • 使用：sudo apt-fast install python3.5 （使用方法就是把apt-get 换成apt-fast）
