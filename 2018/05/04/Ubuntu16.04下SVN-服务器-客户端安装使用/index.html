SVN 服务器搭建

SVN 客户端使用

  • svn checkout svn://192.168.31.44/cxp 检出项目
  • svn co svn://192.168.31.44/cxp 检出项目
  • svn add test.txt 添加txt文件
  • svn add * 添加当前目录下所有文件
  • svn add user 添加user目录(同时将目录下所有内容（递归）添加)
  • svn status 查看文件状态
  • svn commit -m “描述” 提交代码
  • svn up 更新代码
