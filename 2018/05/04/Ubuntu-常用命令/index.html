Ubuntu 常用命令

系统命令

  • 查看系统位数：getconf LONG_BIT
  • 查看磁盘占用：df -h
  • 查看当前系统版本：cat /etc/issue
  • 安装所有依赖：sudo apt-get install -f
  • 查看软件安装目录：whereis java
  • 查询文件运行文件：which java
  • 查询软件进程：ps ax | grep java
  • 卸载xxx软件：sudo apt-get remove xxx
  • 进入到根目录下的user文件夹：cd ~/user
  • root账户设置密码：sudo passwd root
  • 重启：reboot
  • 关机：poweroff

文件夹命令

  • 创建文件夹：mkdir xxx
  • 创建文件夹及子文件夹：mkdir -p xxx/xxx
  • 复制：cp /xx/xx /xx/xx
  • 移动：mv /xx/xx /xx/xx
  • 删除文件夹及子文件：rm -rf /xxx/xxx

文件命令

  • touch cxp.txt 创建文件
  • cat cxp.txt 查看文件内容

解压命令

  • .tar.gz 和 .tgz 解压：tar zxvf filename.tar.gz
  • .tar.bz 解压：tar jxvf filename.tar.bz
  • .tar.z 解压：tar zxvf filename.tar.z
  • .bz2 解压：tar -xvvf filename.bz2
  • .tar.xz 解压：tar xvJf filename.tar.xz
  • .deb安装：sudo dpkg -i deb文件名

防火墙命令

  • 查看端口状态：sudo ufw status
  • 添加80端口：sudo ufw allow 80
  • 开启防火墙：sudo ufw enable
  • 重启防火墙：sudo ufw reload
  • 关闭防火墙：sudo ufw disable
