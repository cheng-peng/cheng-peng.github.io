Ubuntu wine3.0安装

安装依赖

解压wine3.0文件

  • tar xvJf wine-3.0.tar.xz

安装配置

  • ./configure –enable-win64

编译

  • sudo make

安装

  • sudo make install

运行命令检查Wine版本

  • wine64 –version

创建一个wine链接，否则很多程序不能用

  • sudo ln -s /usr/local/bin/wine64 /usr/local/bin/wine

相关配置

  • winecfg
  • 乱码就是缺少字体，复制windows的simsun.ttc字体到～/.wine/drive-c/windows/fonts/目录下
  • sudo cp simsun.ttc ~/.wine/drive_c/windows/Fonts/

  • regedit zh.reg 导入到注册表

  • winecfg

安装TIM

  • sudo apt-get install wine-mono
  • sudo apt-get install wine-gecko
